# CssReset
<Subtitle>
version {version}
</Subtitle>

This package includes a global emotion component that adds page-level styles to establish a common baseline:

\- A CSS reset to standardize default styling of HTML5 elements across browsers, using the popular Eric Meyer Reset

\- @font-face declarations to load the Public Mobile web fonts

\- Set box sizing to border-box to make it easier to size elements

\- Default body styles to establish the base font settings

## Usage

* To install `CssReset` package with npm run terminal:

```jsx
npm i @pm-kit/css-reset

```

Import `CssReset` somewhere at the top level of your app and add it to the render

```jsx
import CssReset from '@pm-kit/css-reset'

const App = () => {
  return (
    <div>
      <CssReset />
      {children}
    </div>
  )
}
```
